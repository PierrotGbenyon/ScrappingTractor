# Projet de scraping du site de tracteurs

# Analyse exploratoire du marché des camions tracteurs

## 1. Contexte du projet

Dans le cadre de ce projet, nous avons réalisé une analyse exploratoire du marché des camions tracteurs à partir de données collectées sur un site spécialisé dans la vente de véhicules industriels.

L'objectif principal était d'extraire automatiquement les informations essentielles des annonces afin de constituer une base de données exploitable pour une première étude descriptive du marché.

## 2. Données exploitées

## 2. Données exploitées

Les variables utilisées sont les suivantes :

| Variable | Description | Utilité dans l'analyse |
|----------|-------------|------------------------|
| manufacturer | Marque ou constructeur du camion | Identifier les constructeurs les plus représentés |
| price | Prix du véhicule | Étudier la distribution des prix et calculer des indicateurs globaux |
| Mileage | Kilométrage du véhicule | Étudier la relation entre kilométrage et prix |
| Year of construction | Année de construction du véhicule | Analyser l'évolution du prix moyen selon l'année |

## 3. Objectifs de l’analyse

- Identifier les marques les plus présentes dans les annonces ;
- Analyser la distribution des prix des camions tracteurs ;
- Étudier la relation entre le kilométrage et le prix ;
- Observer l’évolution du prix moyen selon l’année de construction ;
- Construire un tableau de bord synthétique regroupant les principaux résultats.

## 4. Analyses réalisées

### 4.1 Top 10 des constructeurs

Une visualisation en barres permet d'identifier les constructeurs les plus représentés dans les annonces collectées. Cette analyse donne une vue d’ensemble de la structure de l’offre observée.

### 4.2 Distribution des prix

Un histogramme est utilisé pour représenter la répartition des prix des véhicules. Cette visualisation permet d'identifier les plages de prix les plus fréquentes et d'observer la dispersion globale du marché.

### 4.3 Relation entre kilométrage et prix

Un nuage de points permet d'examiner la relation entre le kilométrage et le prix des camions. Cette analyse vise à repérer une éventuelle tendance entre l'usure du véhicule et sa valeur marchande.

### 4.4 Prix moyen selon l'année de construction

Une courbe représente l'évolution du prix moyen en fonction de l'année de construction. Cette analyse permet d’observer l’effet de l'ancienneté sur le niveau de prix moyen.

### 4.5 Tableau de bord final

Le travail se termine par la création d'un tableau de bord regroupant :

- le prix médian ;
- le prix moyen ;
- le kilométrage moyen ;
- les quatre graphiques produits précédemment.

## 5. Limites de l’analyse

L'analyse présentée est une analyse descriptive. Elle ne permet pas, à elle seule, d'établir des relations causales.
